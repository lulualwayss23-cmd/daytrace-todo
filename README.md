# Morandi Todo

把今天过得清楚一点。

Morandi Todo 是一个简洁、柔和的个人时间管理网页应用，将日计划、专注计时、习惯养成和成长记录放在一起。

## 主要功能

- 添加、编辑和完成日计划
- 正向计时、倒计时和不计时三种任务模式
- 专注计时滴答白噪音与完成提醒
- 为习惯设置每日目标并记录投入时间
- 日计划完成后同步专注时间到关联习惯
- 周、月、年习惯完成记录
- 今日、本周和本月数据统计
- 自动生成并复制本周总结
- 历史日期任务浏览
- 浏览器本地保存数据
- 适配电脑和手机屏幕

## 使用方法

直接使用浏览器打开 `index.html` 即可。

如果需要让同一局域网中的手机访问，可以在 Windows 电脑上运行 `start-mobile.cmd`，然后按照窗口中的提示使用手机浏览器访问。

## 数据说明

任务和习惯数据保存在浏览器的 `localStorage` 中，不会自动上传到 GitHub，也不会在不同浏览器或设备之间同步。

清除浏览器网站数据可能会删除记录。建议定期使用应用内的数据导出功能进行备份。
